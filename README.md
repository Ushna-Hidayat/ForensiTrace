# ForensiTrace v2.0

A digital forensic investigation system built to support evidence collection, analysis, and reporting — from a single tool instead of stitching together a dozen different utilities.

## Overview

ForensiTrace v2.0 is a Python-based forensics platform with 18 integrated modules covering the core stages of a digital investigation: acquiring evidence, analyzing artifacts, and generating a final report.

## Modules

- **Dashboard** — central overview of the active case
- **Timeline Analyzer** — reconstructs event sequences from system data
- **File / Image Metadata** — extracts metadata from files and images
- **Hash Verifier** — verifies file integrity via hash comparison
- **Browser History** — parses and analyzes browser activity
- **Email Analyzer** — examines email artifacts
- **Event Log Parser** — parses Windows event logs
- **Steganography** — detects hidden data within files
- **USB Monitor** — tracks USB device connection history
- **Memory Dump** — analyzes memory dump files
- **Deleted Files** — recovers deleted file artifacts
- **Password Checker** — evaluates password strength/security
- **Packet Analyzer** — inspects captured network traffic
- **File Integrity Monitor** — tracks file changes over time
- **Malware Hash Lookup** — checks file hashes against known malware signatures
- **Keyword Search** — searches evidence for specific terms
- **Charts** — visualizes findings and case data
- **Generate Report** — compiles findings into a final investigation report

## Tech Stack

- **Language:** Python
- **GUI:** Tkinter
